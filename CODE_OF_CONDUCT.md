# 行为准则

Norn 希望维护一个友善、专业、尊重差异的开源社区。

## 我们鼓励

- 尊重不同经验、背景和观点
- 针对问题和代码讨论，避免针对个人
- 给出具体、可执行、善意的反馈
- 承认错误并积极修正
- 关注社区和项目的长期利益

## 不可接受的行为

- 骚扰、歧视、侮辱、威胁或人身攻击
- 发布他人的隐私信息
- 持续扰乱讨论、Issue 或 Pull Request
- 不受欢迎的性化表达或内容
- 其他在专业协作环境中明显不当的行为

## 执行

维护者可以编辑、隐藏或删除不当内容，并视情况警告、暂时限制或永久禁止参与者。严重或敏感问题请通过仓库可用的私密渠道联系维护者，例如 [GitHub 私密安全报告](https://github.com/duyl328/norn/security/advisories/new)。

参与本项目即表示同意遵守本准则。
