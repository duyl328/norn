# Norn

轻量代码 / Git 工作台。

> 本仓库仅用于**分发安装包与自动更新**,不包含源代码。

## 下载

前往 **[最新 Release](https://github.com/duyl328/norn/releases/latest)** 下载:

| 平台 | 文件 |
| --- | --- |
| macOS (Apple Silicon) | `Norn_x.y.z_aarch64.dmg` |

## 安装

1. 打开 `.dmg`,把 **Norn** 拖入「应用程序」。
2. 首次打开如提示「无法验证开发者 / 已损坏」,前往
   **系统设置 → 隐私与安全性 → 仍要打开**(macOS 15 之前可右键 → 打开)。

## 自动更新

应用内置自动更新:启动时静默检查,有新版会弹窗询问;也可在菜单 **检查更新** 手动触发。
更新通过本仓库的 Release 分发,验签后安装,无需重新下载 dmg。

## 文件关联

安装后可在文本/源码文件上「右键 → 打开方式 → Norn」,并选「全部更改」设为默认。
支持 txt、md、json、yaml 及上百种常见源码扩展名。
